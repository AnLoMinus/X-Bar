GitHub Projects Viewer

הצג את הפרויקטים של GitHub בצורה מרשימה באתר שלך

🚀 מדריך התקנה

1. הוספת הסקריפט

הוסף את הקוד הבא לפני תג הסגירה של ה-body:

<script src="https://cdn.jsdelivr.net/gh/anlominus/X-Bar/github-projects-widget.js"></script>

2. הגדרת התצוגה

הוסף את הקוד הבא למקום בו תרצה להציג את הכפתור:

<div id="github-projects-widget"></div>

3. התאמה אישית

ניתן להתאים את התצוגה באמצעות הגדרות נוספות:


const widgetConfig = {
    username: 'your-github-username',
    theme: 'dark', // או 'light'
    maxProjects: 10,
    showLanguages: true,
    showStars: true
};
